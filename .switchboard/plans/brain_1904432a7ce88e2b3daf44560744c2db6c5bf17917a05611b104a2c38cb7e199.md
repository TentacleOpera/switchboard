# Deprecate Heap SDK from codebase

**MISSING INFORMATION REQUIRED BEFORE EXECUTION:** 
*   **Data Parity Confirmation:** This plan requires explicit confirmation that the live QA data stream task was successful and Amplitude data matches expected metrics.

## Step 1: Understand the Goal
Remove the legacy Heap Analytics SDK (`@heap/react-native-heap`) from the React Native application. This will reduce bundle size, eliminate tracking debt, and speed up compilation times.

## Step 2: Complexity, Edge-Case & Dependency Audit
*   **Complexity:** Low, but sweeping. It touches root configuration files (`package.json`, `app.json`, `babel.config.js`) and root components (`App.tsx`).
*   **Edge Cases:** If the Babel auto-instrumentor plugin is removed from `package.json` but mistakenly left in `babel.config.js`, the Metro bundler will crash instantly on startup.
*   **Dependencies & Conflicts:** Cannot be deployed until the `Integrate Amplitude SDK into app` step has been pushed to production and verified.

## Step 3: Improve Plan (/improve-plan)
*   **Grumpy Persona:** Removing the NPM package is easy, but you have to rip out the `HeapNavigationContainer` in `App.tsx` and all the manual `Heap.identify()` calls, otherwise TypeScript will fail to compile.
*   **Balanced Persona:** We will systematically strip Heap out of `package.json`, `app.json`, `babel.config.js`, and `App.tsx`. For the scattered manual calls, the engineering team must run a global regex to strip them, but the critical root bindings are specified below.

## 5. Exhaustive Implementation Spec

### Adversarial Synthesis
#### Grumpy Critique
If you leave `./node_modules/@heap/react-native-heap/instrumentor/src/index.js` in `babel.config.js` while the `node_modules` folder no longer has the package, Metro will throw an unrecoverable "Module not found" exception.
#### Balanced Response
We will execute an exact search and replace on `babel.config.js` to guarantee the instrumentor plugin is safely decoupled from both the default and production environment build arrays.
