# Align Controls Strip Project Button Sizes

The project controls buttons in `kanban.html`'s controls strip (Add Project `+`, Delete Project, and Copy Chat Prompt) currently render inconsistently:
- The Add Project button (`.btn-add-plan`) is physically configured to `18px x 18px` in CSS.
- The Delete Project and Copy Chat Prompt buttons (`.strip-icon-btn`) are physically configured to `20px x 20px` in CSS.
- The Delete Project and Copy Chat Prompt buttons have a transparent border and a dark background, making them appear smaller (floating) and harder to click compared to the brightly styled Add Project button.
- The neighboring inputs/buttons (select dropdown and ASSIGN button) are approximately `24px` tall.

This change will normalize the sizes of these three control buttons to `20px x 20px` and style the delete and copy prompt buttons with matching borders and hover states to align visually.
