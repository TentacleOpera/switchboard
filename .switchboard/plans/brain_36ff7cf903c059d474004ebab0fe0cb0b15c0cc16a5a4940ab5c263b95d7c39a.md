# Update Switchboard Status Bar Button Label to 'Agents' and Update Icon

## Goal

Change the status bar button that opens the agent terminals from its current confusing label "Grid" (with the inappropriate icon `$(split-horizontal)`) to "Agents" (with a matching icon like `$(terminal)` or `$(hubot)`). Additionally, update all related configuration descriptions in `package.json` and the setup UI in `setup.html` to reflect this naming.

### Problem Analysis & Background
- **Core Problem**: The status bar button to open agent terminals is currently labeled "Grid" with a `$(split-horizontal)` icon. This label is confusing because it suggests a window layout grid operation rather than launching the agent CLI terminals.
- **Root Cause**: The name "Grid" refers to the internal layout mechanism (terminal grid), which does not match user-facing concepts of AI "Agents" terminals.
- **Proposed Solution**: 
  - Change the label to "Agents".
  - Change the icon to a more appropriate icon such as `$(terminal)` (matching standard terminal buttons next to it: Clear, Reset) or `$(hubot)` (representing agents).
  - Update related text in `package.json` and `setup.html` that describes the status bar buttons.

## User Review Required

The user has selected the **`$(hubot)`** (robot head) icon for the "Agents" button. We will implement this choice.
