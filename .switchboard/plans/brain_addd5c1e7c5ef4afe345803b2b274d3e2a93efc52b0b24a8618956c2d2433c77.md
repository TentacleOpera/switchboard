# Fix Collapse Sidebar Button inside Online Docs Tab

## Goal
Fix the collapse sidebar button in `planning.html` for the "Online Docs" tab. Currently, clicking it does not do anything because the JavaScript is missing the logic to apply the sidebar state to the `online` tab.

## Proposed Changes

### Webview

#### [MODIFY] [planning.js](file:///Users/patrickvuleta/Documents/GitHub/switchboard/src/webview/planning.js)
Update the sidebar state management to include the `online` tab alongside the `local` and `research` tabs:
1. **Initialize state**: Apply the sidebar state to the `online` tab on load.
2. **Toggle state**: Ensure `applySidebarState('online', state.docsListCollapsed)` is called when toggling the sidebar.
3. **Tab changes**: Call `applySidebarState('online', state.docsListCollapsed)` when the "Online Docs" tab becomes active.
