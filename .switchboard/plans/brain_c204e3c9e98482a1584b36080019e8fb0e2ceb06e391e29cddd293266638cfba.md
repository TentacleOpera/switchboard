# Add Complexity Rating Instruction to Antigravity Chat Prompt

Add a complexity assessment instruction to the initial Antigravity Consultation & Planning Mode prompt (located in `DEFAULT_CHAT_BASE_INSTRUCTIONS` and `switchboard-chat.md`). This ensures that when the agent writes an `implementation_plan.md`, it automatically assesses the task complexity (on a 1-10 scale) and writes this information into the `## Metadata` section. This metadata is parsed by the Switchboard Kanban board to categorize and route the card correctly.

## Goal

### Core Problems & Background Context
Currently, the Consultation & Planning Mode prompt defines rules for writing plans (such as enforcing specific H1 titles and restricting folder/file locations) but does not mention the `## Metadata` section or the `**Complexity:**` rating. Consequently, when an Antigravity agent generates an implementation plan, the plan's complexity defaults to "Unknown" in the Switchboard Kanban board unless the user manually specifies or overrides it. Adding a complexity assessment rule will allow agents to assess and document the complexity score directly in the plan from the beginning, which the Kanban board will then parse automatically.
