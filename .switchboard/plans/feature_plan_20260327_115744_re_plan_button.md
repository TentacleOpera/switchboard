# Re-plan button

## Goal

Change the function of the current 'context map' button in the planner kanban column to be 'Re-plan' instead. This should simply send the planning prompt for the selected cards to the planner agent.

Use case: The user creates a plan, and then uses a free lower-tier model to bulk out the plan via the copy prompt button in the new column. However, they still want a high reasoning model to review the plan before sending it off, so they click the Re-plan button to do this.

The icon does not need to chamge, it suits this function more.
