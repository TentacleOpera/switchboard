# Implement Word Wrap for ClickUp/Linear Sidebar Tickets

This plan removes CSS constraints preventing text wrapping in ClickUp/Linear ticket displays in the sidebar, allowing all text elements to wrap indefinitely.

## Problem Analysis

Located in `/Users/patrickvuleta/Documents/GitHub/switchboard/src/webview/implementation.html`, the ClickUp/Linear ticket cards have multiple CSS classes with `white-space: nowrap`, `overflow: hidden`, `text-overflow: ellipsis`, and `-webkit-line-clamp` constraints that prevent text from wrapping. This causes long ticket titles and descriptions to extend horizontally instead of wrapping to multiple lines.

## Affected CSS Classes

The following classes in implementation.html need modification:

1. **`.project-issue-title`** (lines 459-463)
   - Current: `white-space: nowrap`, `overflow: hidden`, `text-overflow: ellipsis`
