# Fix Kanban Column Card Ordering

Change the `getPlansByColumn` method to sort cards by `updated_at DESC` instead of `ASC` so that cards in each column are displayed in descending timestamp order (newest first), matching the behavior of other query methods.

## Root Cause

Line 904 in `src/services/KanbanDatabase.ts` uses `ORDER BY updated_at ASC` in the `getPlansByColumn` query, while all other similar methods (`getBoard`, `getBoardFiltered`, `getCompletedPlans`) use `ORDER BY updated_at DESC`.

## Fix

Change line 904 from:
```sql
ORDER BY updated_at ASC
```
to:
```sql
ORDER BY updated_at DESC
```

- `src/services/KanbanDatabase.ts` - Line 904
