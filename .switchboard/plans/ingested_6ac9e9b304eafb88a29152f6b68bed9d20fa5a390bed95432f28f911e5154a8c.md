# Add Import Buttons to ClickUp/Linear Sidebar Cards

Add an "IMPORT" button directly on ClickUp task cards in the sidebar, matching the existing Linear card pattern. Also remove the redundant "SEND TO PLANNER" button from Linear cards since it's unnecessary on card level.

## Changes

### 1. Add Import Button to ClickUp Cards
- **File:** `src/webview/implementation.html`
- **Function:** `renderSidebarClickUpProjectList` (around line 3896-3912)
- Add IMPORT button to the card HTML template using existing `project-issue-import-btn` CSS class
- Add `data-clickup-import-id` attribute for event handling

### 2. Update ClickUp Card Event Handler
- **File:** `src/webview/implementation.html`  
- **Function:** Event listener around line 3919-3924
- Check for import button clicks before navigating to task details
- Send `linearImportTask` message (note: this message type handles both ClickUp and Linear)

### 3. Remove Send to Planner from Linear Cards
- **File:** `src/webview/implementation.html`
- **Function:** `renderSidebarLinearProjectList` (around line 3305-3316)
- Remove the "SEND TO PLANNER" button HTML (keep IMPORT button)
- Remove the `project-issue-send-planner-btn` click handler from the issuesContainer event listener (around line 4183-4193)

## Verification
- [ ] ClickUp cards show green IMPORT button
- [ ] Clicking IMPORT on a ClickUp card imports the task without opening detail view
- [ ] Linear cards no longer show SEND TO PLANNER button
- [ ] Detail view buttons remain unchanged for both providers
