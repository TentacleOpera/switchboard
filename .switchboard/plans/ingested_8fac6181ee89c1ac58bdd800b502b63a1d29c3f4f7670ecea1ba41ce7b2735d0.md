# Relax Plan Validation for Additional Plan Folder

Relax the strict plan format validation for the additional plan folder to accept any markdown file, using the first H1 heading as the title.

## Changes Required

### 1. TaskViewerProvider.ts - Update `_isLikelyPlanFile`

Add a parameter to distinguish between the default plans folder and the additional plan folder. When processing the additional folder, skip all format checks and return `true` for any `.md` file.

**Location:** `src/services/TaskViewerProvider.ts:9447-9474`

**Current logic:**
- Requires H1 header
- Requires specific section headers OR complexity/tags metadata
- OR must be named `implementation_plan.md`

**New logic:**
- Add `isAdditionalFolder: boolean` parameter
- If `isAdditionalFolder === true`: return `true` for any `.md` file
- Otherwise: keep existing strict validation

### 2. TaskViewerProvider.ts - Update `_syncConfiguredPlanFolder`

Pass `isAdditionalFolder: true` when calling `_isLikelyPlanFile` for the additional plan folder.

**Location:** `src/services/TaskViewerProvider.ts:7595`

### 3. TaskViewerProvider.ts - Add title extraction helper

Add a function to extract the first H1 heading from a markdown file for use as the plan title.

**Location:** New function in `TaskViewerProvider.ts`

```typescript
private _extractFirstH1(content: string): string | null {
    const match = content.match(/^#\s+(.+)$/m);
    return match ? match[1].trim() : null;
}
```

### 4. TaskViewerProvider.ts - Update plan creation logic

When mirroring files from the additional folder, use the extracted H1 as the title if available. If no H1 exists, use the filename.

**Location:** `src/services/TaskViewerProvider.ts:7612-7620`

### 5. setup.html - Add informational note

Add a note in the Database Operations section explaining that any markdown file will be accepted and titles are extracted from the first H1 heading.

**Location:** `src/webview/setup.html:619-622`

**Add after the input field:**
```html
<div style="font-size:9px; color:var(--text-secondary); margin-top:4px; line-height:1.3;">
    Accepts any .md file. Plan title is extracted from the first H1 heading (falls back to filename).
</div>
```
