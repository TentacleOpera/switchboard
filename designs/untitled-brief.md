# Switchboard landing page

Create a premium, high-fidelity developer-dark marketing landing page for "Switchboard", a 0→1 project tool that runs complex builds around coding agents. 

Design Aesthetics & Color Palette:
- Background (Canvas): Sleek near-black (#0B0D10).
- Surfaces: Slightly raised, dark-gray cards and sections (#14171C).
- Hairline Borders: Sharp, subtle separator borders (#232830).
- Accent Color: Electric blue (#5B8CFF) used sparingly for buttons, links, kickers, and focus rings to create a high-contrast terminal-like feel.
- Typography: Use "Inter" (or a clean, modern sans-serif) for body and headlines. Use a crisp monospace like "JetBrains Mono" or "IBM Plex Mono" for kickers, captions, metrics, and navigation elements. 

Layout and Structure (Single-Page Long Scroll):

1. Navigation Bar (Sticky & Minimal):
   - Transparent background transitioning to a solid dark surface on scroll.
   - Left: "Switchboard" wordmark in clean monospace.
   - Right: Links for "Demo", "Features", "GitHub", and a primary accent button for "Install".

2. Hero Section:
   - Eyebrow (Monospace): "THE 0→1 PROJECT TOOL" in electric blue.
   - Headline: "Run the whole build — research to shipped — from one control plane." (Bold, clean typography).
   - Subtitle: "Switchboard is the control plane for your entire 0→1 build — research, prototypes, plans, and the coding agents that execute them. Not another agent orchestrator. The tool that runs the project around them."
   - Action Buttons: A primary CTA button "See it in action ↓" (smooth-scroll behavior) and a secondary outline CTA "Install for VS Code".
   - Hero Asset: A large, elegant 16:9 mockup container showing a project/feature hierarchy board (representing the core workspace board).

3. "The Shift" Section (Short, high-impact band):
   - Headline/Kicker: "THE HARD PART MOVED" in monospace.
   - Short, punchy paragraph explaining that code generation is now cheap, but project organization, plan tracking, and repo alignment are what's hard.

4. Feature Showcase (Five Columns/Rows alternating left/right layout):
   - Show 5 pillars, each containing: Kicker (monospace, accent colored), Headline (bold, clean), 2-3 line body, and a corresponding card containing a 16:9 demo video placeholder.
     - Pillar 1: "PROJECT MANAGEMENT" — "A project model deep enough for real work."
     - Pillar 2: "ANY AGENT, ONE BOARD" — "Drive every coding CLI from one place."
     - Pillar 3: "WHEREVER YOU WORK" — "Runs inside your tools. Drivable from anywhere."
     - Pillar 4: "ARTIFACTS, FIRST-CLASS" — "Where a 0→1 tool separates from an orchestrator."
     - Pillar 5: "BUILT TO BE A CONTROL PLANE" — "Decoupled from your repo. Ready for your team."

5. Founder Pull-Quote Section:
   - A quiet, elegant testimonial block with a thin vertical border on the left:
     "I built Switchboard by adding the things I actually needed to take real work from 0 to 1 — research, plans, team, code. I use it every day."
     — built by a product lead, used daily in production

6. Social Proof / Stats Strip:
   - A clean horizontal bar displaying stats: "4,000+ installs", "Open Source", "Works with Claude Code & Antigravity", "Syncs to Linear & Notion".

7. Final CTA:
   - Headline: "Start your next 0→1 on Switchboard."
   - Buttons: "Install for VS Code" (Primary) and "Star on GitHub" (Secondary).

8. Footer:
   - Tagline: "Switchboard. The 0→1 project tool."
   - Links: GitHub, VS Code Marketplace, License, Docs.

Deliver a visually striking, professional, and responsive layout with generous negative space, smooth transitions, and subtle glowing/fading dot grid elements in the hero background to capture a state-of-the-art developer product landing page.
