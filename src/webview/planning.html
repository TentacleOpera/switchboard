NOTEBOOKLM INTEGRATION
NotebookLM integration allows you to upload all your code into NotebookLM to access unlimited Gemini quota for planning features and diagnosing bugs.
Workspace
1. BUNDLE CODE
Package code into docx files for NotebookLM compatibility.
2. UPLOAD TO NOTEBOOKLM
Create new Notebook and upload all files in the integration folder as sources.
3. COPY SPRINT PLANNING PROMPT
Generate a prompt for NotebookLM to write expanded plans for every plan in the NEW Kanban column. Copy the output and use Step 4 below to import.
4. IMPORT PLANS
Paste your expanded plans from NotebookLM. Plans matching titles in the NEW column will overwrite the originals; others will be created as new plans.
Switchboard Research offers an easy way to use Google AI Studio to quickly perform high quality research, and get the results back into your workspace as a fully formatted document.
STEP 1: DRAFT RESEARCH PROMPT
Copy a meta-prompt to your clipboard. Paste it into your IDE agent to draft a detailed research prompt for Google AI Studio.
Central Question / Topic
STEP 2: RUN RESEARCH IN GOOGLE AI STUDIO
  1. Open Google AI Studio (aistudio.google.com).
  2. Paste the drafted prompt.
  3. Enable Search Grounding for web search access.
  4. Run the prompt and use the response menu at the top right to copy as markdown.
STEP 3: IMPORT RESEARCH DOCUMENT
Import the research findings copied from Google AI Studio.
